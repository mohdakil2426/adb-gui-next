# Emulator Root Verification Fix Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make the emulator root workflow report success only when the AVD actually boots with working Magisk root, while hardening the ramdisk patch pipeline against the rootAVD differences found in the audit.

**Architecture:** Split the current single "root success" signal into two explicit states: patched ramdisk installed, then verified root after a cold boot. Add a backend verification command that checks boot completion, Magisk app availability, and `su -c id -u`; update frontend copy/state so users see "patch installed" until verification passes.

**Tech Stack:** Tauri 2, Rust 2024, React 19, TypeScript 6, Zustand, Vitest, Bun, Android platform-tools, rootAVD reference workflow.

---

## Reference Baseline

- rootAVD GitHub README: `https://github.com/newbit1/rootAVD/blob/master/README.md`
- rootAVD script: `https://raw.githubusercontent.com/newbit1/rootAVD/master/rootAVD.sh`
- Context7 library used for audit: `/newbit1/rootavd`

Relevant rootAVD behaviors to preserve:

- RootAVD patches the ramdisk, pulls the patched image, installs available apps, then shuts down the AVD.
- Magisk root is only usable after the AVD cold-boots with the patched ramdisk.
- Magisk 26+ often needs the FAKEBOOTIMG/manual path for full setup.
- API 30+ ramdisks may contain multiple CPIO archives; rootAVD has extra repack logic before patching.
- Safe-mode and restore paths must remain available when root causes boot problems.

fn adb_shell_checked(app: &AppHandle, serial: &str, cmd: &str) -> CmdResult<String> {
    let wrapped = format!("{cmd}; echo {EXIT_CODE_MARKER}$?");
    let output = adb_shell(app, serial, &wrapped)?;

}

Add this public function to `src-tauri/src/emulator/root.rs`:

```rust
pub fn verify_avd_root(
    app: &AppHandle,
    avd_name: &str,
    serial: &str,
) -> CmdResult<RootVerificationResult> {
    if !runtime::is_serial_online(app, serial) {
        return Ok(RootVerificationResult {
            status: RootActivationStatus::VerificationFailed,
            boot_completed: false,
            su_uid: None,
            magisk_package: None,
            message: format!("AVD '{avd_name}' is not online over ADB."),
        });
    }

    let boot_completed = getprop(app, serial, "sys.boot_completed").unwrap_or_default();
    let boot_ready = boot_completed.trim() == "1";
    if !boot_ready {
        return Ok(RootVerificationResult {
            status: RootActivationStatus::VerificationFailed,
            boot_completed: false,
            su_uid: None,
            magisk_package: detect_root_app_package(app, serial),
            message: "Emulator is still booting. Wait for the home screen, then verify again.".into(),
        });
    }

    let magisk_package = detect_root_app_package(app, serial);
    let su_output = run_binary_command_allow_output_on_failure(
        app,
        "adb",
        &["-s", serial, "shell", "su", "-c", "id -u"],
    )
    .unwrap_or_default();
    let su_uid = su_output.trim().lines().last().map(str::trim).filter(|value| !value.is_empty()).map(ToOwned::to_owned);
    let verified = su_uid.as_deref() == Some("0");

    Ok(RootVerificationResult {
        status: if verified {
            RootActivationStatus::Verified
        } else {
            RootActivationStatus::VerificationFailed
        },
        boot_completed: true,
        su_uid,
        magisk_package,
        message: if verified {
            "Root verified: su returned uid 0.".into()
        } else {
            "Root not verified: su did not return uid 0. Open Magisk Manager, complete Additional Setup if prompted, reboot, then verify again.".into()
        },
    })
}
```

Ensure `RootActivationStatus` and `RootVerificationResult` are imported at the top of the file.

  }
}

fn detect_multi_cpio_ramdisk(app: &AppHandle, serial: &str) -> CmdResult<bool> {
